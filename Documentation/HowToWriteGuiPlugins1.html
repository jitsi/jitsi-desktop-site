How to write GUI plugins

Getting started

This tutorial will guide you through the process of creating a plugin for the Jitsi user interface.

Before starting you should obtain the Jitsi source code. You could obtain it either from the latest release, from CVS, or from nightly builds. Make sure you have compiled the source and you’re ready to start :)
